# Postmortem

## Template

```markdown
## 概要

-  事象の簡単な説明

## 起きたこと

- 何が起こったのか詳細を記載

## 原因

- 事象の根本原因を分析

## 解決策

- どのように解決したかを記載

## 影響

- 影響範囲や被害状況を記載

## 対応者

- 

## タイムライン

| 時間    | 出来事   |
| :---- | :---- |
| HH:MM | 事象発生  |
| HH:MM | 初動対応  |
| HH:MM | 原因特定  |
| HH:MM | 解決策実施 |
| HH:MM | 事象収束  |

## 幸運だったこと

- 

## アクションアイテム

- [ ] 

```
