# メールの今：2025年後半の動向

<Callout type="info">
Claude Deep Research によるアウトプット（2026/02）
</Callout>

2025年8月〜2026年2月は、この10年で最も活発なメールイノベーション期間となった。BillionMail、Mail0（Zero）、Stalwartの3つのOSSプロジェクトがそれぞれGitHub 10,000スター超えを達成。手軽なAI、成熟したRustインフラ、プライバシー意識の高まりが重なり、メールスタックの全レイヤーで革新が起きている。

## AIネイティブなメールクライアント

AIが「機能」ではなく「アーキテクチャ」であるメールクライアントが台頭してきた期間だった。

### Mail0（Zero）
YC支援のOSSクライアント。Next.js + React製で、スレッド要約・返信生成・メールとの自然言語チャット機能を搭載。GitHub 10,400スター、MITライセンス、セルフホスト対応。サイト：0.email

### Filo Mail
メールスレッドからTo-Doを自動抽出しカレンダーと同期。iOS / macOS / Android / Windows対応。Product Huntで約145アップボート。

### April
YC S25。音声だけで通勤中に100通以上のメールを処理できる音声ファーストなメール管理ツール。

### Shortwave
元Google Inboxチームが構築。AIアシスタント「Ghostwriter」がユーザーの文体を学習して返信を下書き。月額$7〜14でSuperhumanの代替として人気。r/productivityでの**Superhuman vs Shortwave論争**は最も活発なスレッドの一つに。

### Superhuman
2025年半ばにGrammarlyが買収。AIメール市場がニッチから主流へ移行したシグナル。

### Inbox Zero
OSSのAIメール管理ツール。ラベリング・アーカイブ・一括購読解除を自動化。Product Hunt #1、GitHub 7,000スター超。OpenAI / Anthropic / Ollamaに対応、Dockerでセルフホスト可能。

## OSSメールインフラの転換点

複数の本格的なOSSメールインフラプロジェクトが同時に、商用ソフト並みの完成度に達した。

### BillionMail
フルメールサーバー（Postfix + Dovecot + Rspamd）＋ニュースレター管理＋メールマーケティングを1つのDockerデプロイで。**8分で設置完了**。GitHub 12,000〜13,400スター、AGPL-3.0ライセンス、Go製。

### Stalwart Mail Server
全てRust製。JMAP / IMAP4 / POP3 / SMTP / CalDAV / CardDAV / WebDAVをシングルバイナリで提供。**LLMによるスパムフィルタリング**も実装。EUのNLnetから資金提供。GitHub 10,600スター。

### Listmonk
v5.0でビジュアルドラッグ&ドロップのメールビルダー追加。控えめなハードウェアで数百万購読者を処理できる定番ツール。Go + Vue.js製、GitHub 18,400スター。

### Mox
シングルバイナリのGo製メールサーバー。DMARC / DKIM / SPF / DANEを自動設定。MITライセンス、NLnet（EU）から資金提供。GitHub 5,200スター。

### KumoMTA
PowerMTAのベテランが構築した大規模送信向けRust製MTA。Luaベースのポリシースクリプティング。数十億通規模のESP向け。

## セルフホスティング＆プライバシー

r/degoogleが約**30万人**に成長。GoogleのGmailデータをGemini AI学習に使用、Microsoftの強制的な新Outlook移行、ポスト量子暗号への意識の高まりが脱出を加速させた。

**Proton Mail**（1億人超）が筆頭だが、**Tuta**（旧Tutanota）はポスト量子暗号を先行実装済み、件名も暗号化、月額€3と優位点がある。新興の**Atomic Mail**は1年未満で100万ユーザー突破。**PurelyMail**は年間$10の格安で熱狂的支持を獲得。

メールエイリアシング（SimpleLogin、addy.io、Firefox Relay、Cloudflare Email Routing）も独立したプライバシー手法として定着しつつある。

## 開発者ツール＆メールマーケティング

**Resend**がJSエコシステムのデファクトスタンダードとして定着。React Email 4.0（リンター・スパムスコア・レスポンシブプレビュー内蔵）をリリース。**Unosend**（廉価代替）、**Loops**（SaaS向け統合マーケティング）、OSS代替の**Postal**など競合が急増。

**Maillayer v2.0**は買い切りモデルのセルフホスト型Mailchimp代替としてProduct Hunt #1（379アップボート）を獲得。Amazon SES使用で1,000通あたり$0.10、月額料金なし。

**beehiiv**は月間30億通処理、55,000+クリエイター。AIウェブサイトビルダーとゼロコミッションのデジタルプロダクト販売を追加し、Substackに直接的な脅威を与えている。

## この半年が示す5つの構造変化

<Steps>
<Step>
**Rustがメールインフラの言語になりつつある**

StalwartとKumoMTAはC言語時代のメールサーバーの世代交代を象徴。メモリ安全性、高パフォーマンス、モダンなアーキテクチャ。両プロジェクトともEU資金を受けており制度的支援もある。
</Step>
<Step>
**「AI搭載」から「AIネイティブ」が標準に**

Mail0やShortwaveは従来のメールクライアントにAIを追加するのではなく、体験全体を言語モデルの上に再構築。米国労働者の**45%がAIを業務利用**（Gallup 2025 Q3）しており、このアプローチは主流への期待に移行中。
</Step>
<Step>
**セルフホストメールが本当にアクセシブルに**

BillionMailの8分Docker設置、Moxのシングルバイナリ、Stalwartのオールインワン。参入障壁は劇的に低下。残る課題はGmail/Microsoftの厳格なフィルタリングに対する配信性だが、デフォルト設定の改善で対処が進む。
</Step>
<Step>
**「買い切りモデル」がメールSaaSを侵食**

MaillayerのProduct Hunt #1とBillionMailの爆発的成長は、サブスク料金を拒否する市場セグメントの存在を示す。AWS SESがメール単価を無視できるレベルにした今、月額課金の正当性が問われている。
</Step>
<Step>
**ポスト量子暗号がもはや理論ではない**

Tutaはすでに実装済み、Protonは展開中。ポスト量子ロードマップを発表していないメールプロバイダは遅れていると見なされるリスクがある。
</Step>
</Steps>
