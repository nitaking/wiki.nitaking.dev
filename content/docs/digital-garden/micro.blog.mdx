### micro.blogとは

{% embed url="https://micro.blog/about" %}

Micro.blogはブログ・ソーシャルメディア・コミュニティを組み合わせたプラットフォームです。

短い投稿もブログとして扱い、それをタイムラインに表示します。

既存のSNSとの違いは、[IndieWeb](https://micro.blog/about/indieweb)・fediverse標準であるかです。特定の会社が中央集権的に囲い込みを行うSNSとは異なり、IndieWebとして以下の特徴を持っています。

> * 独自ドメイン対応
> * Webmenti　on
> * Cross Post / Import
