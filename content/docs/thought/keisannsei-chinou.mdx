# 結晶性知能

> 結晶性知能とは、<mark style="color:yellow;">学校教育や社会経験などによって獲得した知識や経験、正確さと結びついた能力</mark>です。一般常識や言語能力、コミュニケーション能力、業務処理の知識、仕事のスキルなどが該当します。 結晶性知能は、年齢を重ねるごとに強くなっていきます。20歳以降もゆるやかに発達を続け、高齢になっても20歳ごろのレベルを維持できると言われています。 結晶性知能と似た言葉に「[流動性知能](ryudousei-chinou.md)」があります。[流動性知能](ryudousei-chinou.md)は、新しい環境に適応するために、新しい情報を獲得し、それを処理し、操作していく知能です。処理のスピード、直感力、法則を発見する能力などが含まれます。
